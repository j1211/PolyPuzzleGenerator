・概要
線を引くと、ピースと枠穴（複数可）を認識して、表示します。
まだ開発中なので、バグがあるかもしれません。


・開発環境
Visual Studio 2015
Dxライブラリ
Windows10


・使い方
ソースコードをプロジェクトに追加して、ビルド実行するだけ。


・操作
線を引く … マウス左ドラッグ
線を消す … Backspace
縮小／拡大 … Z, X
平行移動   … W, D, S, A

Next(次に進む) … →キー
Back(前に戻る）… ←キー

Escape … 終了
